# Cricket-player--analysis-dashboard
Power BI Cricket Player Analysis Dashboard

## 📊 Project Overview

This dashboard analyzes cricket players based on country, batting hand, and bowling skills using Power BI.

## 📌 Key Insights

* India has the highest number of players
* Majority players are right-handed
* Right-arm medium is the most common bowling style
